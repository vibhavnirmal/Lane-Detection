# Lane-Detection

[![CodeFactor](https://www.codefactor.io/repository/github/vibhavnirmal/lane-detection/badge)](https://www.codefactor.io/repository/github/vibhavnirmal/lane-detection)

Lane Detection using openCV

## Getting Started

Install the dependencies

```bash
$ git clone https://github.com/vibhavnirmal/Lane-Detection.git
$ pip install -r requirements.txt
$ python main.py
```

## Requirements

* [Python 3.X](https://docs.python.org/3/)

Python Package dependencies listed in [requirements.txt](requirements.txt)
